# Even Numbers

## Problem
- print the even numbers from 1 to 99 (inclusive)
- same as exercise 6, only with even numbers instead of odd.

## Examples

```ruby
2
4
6
...
98
```

## Data Structures

- Integers
- Will try with a loop this time

## Algorithm

```ruby
set i = 1

as long as i < 99
  - if i % 2 == 0, print i
  - increment i by 1
end
```
